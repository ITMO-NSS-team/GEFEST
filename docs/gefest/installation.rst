Installation from GitHub
========================
  | git clone https://github.com/ITMO-NSS-team/GEFEST.git
  | cd GEFEST
  | pip install .
  | pytest -s test/

or full installation

  | git clone https://github.com/ITMO-NSS-team/GEFEST.git
  | cd GEFEST
  | pip install .[extra]
  | pytest -s test/

Installation from PyPI
======================
  | pip install gefest  # common
  | pip install gefest[extra]  # full
